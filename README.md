# 命运轮盘 (Turntable)

一款Android 互动转盘应用，支持 AI 智能生成转盘与流程。

## 功能

- **转盘抽取**：加权随机抽取，自定义选项名称、描述、权重
- **流程模式**：多阶段依次抽取，支持自动进行
- **盒子管理**：将转盘归类到盒子，拖拽移动
- **AI 生成**：输入自然语言描述，AI 自动生成转盘或完整多阶段流程
- **AI 编辑**：AI 可修改流程名称、描述、阶段顺序、关联转盘
- **AI 总结**：流程完成后，AI 根据抽取记录生成故事或总结
- **流程结算**：查看完整抽取记录，一键生成 AI 总结

## AI 配置

支持 OpenAI 兼容 API（DeepSeek / OpenAI / Ollama 等）。

设置页 → 填写 API Base URL、模型名称、API Key → 保存后即可使用 AI 功能。
